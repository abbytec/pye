- 80+ comandos (incluidos juegos y sistemas de economía, reputación de usuarios, moderación, y otras utilidades)

- Sistemas automatizados de Foros (con respuestas automáticas con IA), Meme del día y Starboard

### Más info

Breve historia: Hace tiempo había un bot llamado `pye-helper` / `pye-chan` que se usaba para el sv de Discord **Programadores y Estudiantes**. 
Este estaba programado en js, tenia algunos juegos y comandos de moderación.
Hice una migración a ts y lo convertí en un bot modular, agregando multiples integraciones de IA, juegos, mecanismos avanzados de seguridad y auditoría, además de comandos nuevos.
